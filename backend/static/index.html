Express Routes

Users

HTTP Method PATH Description
POST /users/add Create a new user with authorization
POST /user/login Login an existing user
GET /users Get all users
GET /user/:id Get a specific user by ID
PATCH /user/edit/:id Edit a specific user by ID
DELETE /user/delete/:id Delete a user by ID with authorization

Investments

HTTP Method PATH Description
POST /invest/add Add an investment with authorization
GET /investments Get all investments
GET /invest/:user_id Get a specific investment by user ID
PATCH /invest/edit/:user_id Edit a specific investment by user ID with authorization
DELETE /invest/delete/:user_id Delete a specific investment by user ID
